ProcessingBGE references

Here below you will find the complete list of methods available in the library.
Some of them accepts different sets of arguments, their position is very important when you use the methods.

Some methods are flagged as private. Obviously, you can use them, but side effects may appears.
Please check the code first if you want to use them.
